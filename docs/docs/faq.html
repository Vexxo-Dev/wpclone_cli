FAQ

Frequently asked questions about wpclone

Is wpclone production ready?

It is usable for real migrations, but always run dry-run first and validate on staging before production.

Can I migrate files only?

Yes.

Can I migrate database only?

Yes.

Can I skip URL replacement?

Yes.

Where are logs saved?

Default is ./wpclone.log unless changed with WPCLONE_LOG.
