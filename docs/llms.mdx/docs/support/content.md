# Support and Discussions (/docs/support)



## Report issues [#report-issues]

Open an issue with:

* exact command used (redacted)
* local OS and Bash version
* relevant `wpclone.log` output
* source and destination environment details

Issue tracker:

[https://github.com/Vexxo-Dev/wpclone\_cli/issues](https://github.com/Vexxo-Dev/wpclone_cli/issues)

## Discussions [#discussions]

Use Discussions for:
