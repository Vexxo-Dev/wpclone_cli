## Report issues

Open an issue with:

- exact command used (redacted)
- local OS and Bash version
- relevant `wpclone.log` output
- source and destination environment details

Issue tracker:

https://github.com/Vexxo-Dev/wpclone_cli/issues

## Discussions

- feature ideas
