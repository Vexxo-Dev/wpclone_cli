# wpclone Documentation

wpclone is a Bash CLI for migrating WordPress sites between servers.

## What you can do

- Migrate files using rsync over SSH
- Export and import database with WP-CLI or mysqldump/mysql
- Replace URLs safely with wp search-replace
- Run dry-runs before making changes
